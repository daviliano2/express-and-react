# express-and-react

How to run it:
  - go to `react-backend` folder.
    - execute `npm install`
  - run `PORT=3001 node bin/www`
 
  - open a new terminal or a tab in your current terminal
  - go to `client` folder inside of `react-backend`
    - run `npm install`
    - run `npm start`
